### Рекомендуемые правила разметки в Workbook.txt
```
---------------------
* * * Категория * * *
---------------------

\\\\\\\\\\\\\\\\
1.   Раздел
\\\\\\\\\\\\\\\\

#####################
* 		Подраздел
#####################

#* Понятие
  = определение
  - свойство
    #. Описание
      #- Доп. информация
    1. Перечисление
    > код
    # Пример:

# Пример/доп. информация с низким приоритетом
#! Предостережение
#: Ключевая информация
```
